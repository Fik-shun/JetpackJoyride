# JetpackJoyride

Start the game by: python3 game.py

# Requirements

Numpy
Coloroma

# Game

Filled with coins, obstacles, a magnet!
At the end, we've the ultimate boss, the Dragon!
Defeat it to win!

# Controls

W,S,A,D : for movement
F : Firing
C : Speed Boost
Space: Shield Activation(for 10s) 
Q : Quit

# Features

Beautiful ASCII Art
